# LiveKit Signaling Server

A production-ready WebRTC signaling server for 360° meeting platform with LiveKit integration.

## 🚀 Features

- **WebSocket Signaling**: Real-time peer discovery and room management
- **LiveKit Integration**: Automatic JWT token generation
- **Room Management**: Create, join, leave rooms with participant tracking
- **Host Transfer**: Automatic host reassignment when host leaves
- **Health Monitoring**: Built-in health check endpoints
- **Production Ready**: Docker support, Railway ready, Vercel compatible

## 📋 Prerequisites

- Node.js 18+ or Docker
- LiveKit Cloud account (free tier available)
- Railway/Render account for deployment (optional)
